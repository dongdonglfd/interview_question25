#### 1.西邮 Linux 20 周年时间机

时间机中藏着隐藏的彩蛋，通过指针偏移，你可以发现秘密部分。请观察指针操作并预测输出。

```c
char* secret_part() {
    static char str[] = "XiyouLinux20";
    return str + 5;  // 返回指针偏移
}

int main() {
    char *p = secret_part();
    printf("时间机发现隐藏彩蛋: %s\n", p);
    return 0;
}
```

Q1. 解释 str + 5 为什么会返回从第6个字符开始的指针 40%

Q2. 讲讲静态数组和指针的关系 35%

Q3. 解释为什么输出的是字符串剩余部分，而不是单个字符 25%

```
A1：
表达式 str + 5 是指针偏移运算。str 是数组首元素的地址，+5 后指针指向数组中第6个元素（索引从0开始），所以返回的是 "Linux20" 的首地址。这个偏移不会改变原数组，只是让指针从新的位置开始访问。

A2：
静态数组存储在静态区，函数返回静态数组首地址是安全的，因为它不会在函数返回后被销毁。指针可以指向静态数组的任意位置，从而访问数组内部的子串。

A3：
printf("%s", p) 会从指针 p 所指向的位置开始输出，直到遇到 '\0' 结束符。因为指针从第6个字符开始，所以输出的是剩余字符串 "Linux20"，而不是单个字符。
```
